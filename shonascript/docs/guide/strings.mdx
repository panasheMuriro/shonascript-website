# Mashoko

:::caution Zvichiri kunyorwa
:::

## Mashoko chii?
Mashoko zvinhu zvese zvakaiswa mu ma _quotes_ `""` sokuti: 

1. `"Inini ndinonzi Panashe"`
2. `"Ndafara nekukuziva"`

<Question text='"2021" i nhamba  here kana mashoko?'/>
 

`"2021"` aiswa muma _quotes_ saka mashoko haisi `nhamba`.
`2021` inhamba, `"2021"` `mashoko`.
Izvi zvakakosha kuziva pauchazoverenga nezve _topic_ ye Maths kumberi uku.


### Kutsemura Mashoko
Kana uine mutsara wakanzi 

```
"Inini ndinonzi Panashe"
```
Unogona kutsemura mashoko obva wagadzira list uchishandisa `tsemuraMashoko()` seizvi


:::success Ita
1. mu file rakanzi `index.shona`, nyora kuti:
```
mustara = "Inini ndinonzi Panashe"
listReMashoko = mutsara.tsemuraMashoko()
nyora(listReMashoko)
```
2. mu `terminal` nyora kuti `node shona.js`, wobaya `enter`
3. mu `terminal` zvekare, nyora kuti `node index.js`, wobaya `enter`
:::

Ukaita izvi zvakanaka, unofana kuona mu `terminal` makanzi

```
["Inini", "ndinonzi", "Panashe"]
```
Izvi zvakanaka kuziva nekuti zvichazokubatsira kumberi.

### Kutsemura Mavara mumashoko

Unogona kutsemura mavara (letters) ari mumashoko uchishandisa `tsemuraMavara()`

:::success Ita
1. mu file rakanzi `index.shona`, nyora kuti:
```
zita = "Panashe"
listReMavara = zita.tsemuraMavara()
nyora(listReMavara)
```
2. mu `terminal` nyora kuti `node shona.js`, wobaya `enter`
3. mu `terminal` zvekare, nyora kuti `node index.js`, wobaya `enter`
:::


Ukaita izvi zvakanaka, unofana kuona mu `terminal` makanzi:

```
["P","a","n","a","s","h","e"]
```
