## Ma list chii?

Kana iwewe uchida kuenda kunotenga zvinhu zvakawanda, unogona kunyora zvinhu zvaukuda kunotenga 
pa pepa kana mu `phone`. Kana uchinge waita izvi, tinoti wanyora `list` rezvaukuda kunotenga.

<Question text="Ko computer inoona sei list" />

## Manyorerwo e list

`Computer` inoziva kuti wanyora `list` kana ukanyora ma `square brackets` se izvi: 
```
["Muvhuro", "Chipiri", "Chitatu"]
```


:::info **Tsananguro**

Apa ndagadzira `list` re mazuva 3 kubva pana Muvhuro kusvika pana Chitatu. Mu `list` randagadzira iri, mune zvinhu 3. zvinhu izvi zvandaisa mu `list` aya mashoko ane ma `quotes` akapatsanurwa ne ma `comma,`.
:::

:::success Zvii zvinobvumidzwa mu list?
1. `Mashoko` ane ma _quotes_
- sokuti: `["Muvhuro","Chipiri","Chitatu"]`
2. Ma `nhamba`
- sokuti:  `[2020, 2021, 2022]`
3. Ma `variables`
- sokuti:
```
muvhuro = "Muvhuro"
chipiri = "Chipiri"
chitatu = "Chitatu"

[muvhuro, chipiri, chitatu]
```
- Ukatarisa apa, ndaisa ma `variable` mu `list`, asi ndatanga ndamagadzira ma `variable` iwayo se `mashoko` zvakanaka. Kana usina kuzvinzwisisa izvi, 
unofana kudzokera kunoverenga nezve ma `variables` kumashure kune _topic_ yema `variables` kuti zvii zvikuitika.
:::


## Mashandisirwo e list

Kazhinji munhu anogadzira `list` achida kuzorishandisa. Anogona kuda kudzima zvinhu mu`list` imomo, kana kuwedzera zvimwe zvinhu. Kuti ugone kuzoshandisa `list` raunenge wagadzira, unofana kuripa zita. 
Unopa `list` zita kunge uri kugadzira `variable`.
Ngatimboti `list` randagadzira kumashure `["Muvhuro", "Chipiri", "Chitatu"]` waakuda kuripa zita rekuti `mazuva`, unoita se izvi

```
mazuva = ["Muvhuro", "Chipiri", "Chitatu"]
```
:::danger Zita Re list

Zita re list rinonyorwa semagadzirirwo evariable. Manyorero anoita ndeaya: 
1. Zita ngariite shoko rimwe chete rakabatanidzwa
- sokuti `listRangu`
- usati `list Rangu`, nekuti aya mashoko maviri
2. Zita ngarisaite `nhamba`
- sokuti `2021 = ["Muvhuro", "Chipiri", "Chitatu"]`
:::

### Kutora chinhu kubva mu `list`

Vanhu vari mukombi yako iyi vakagara pamasiti emukombi. Masiti aya ane ma `nhamba`. Vanhu vanowanzotangira kuverenga ma`nhamba` pana `1`, asi `computer` inotangira pana `0` nguva dzese
:::danger `Computer` inotangira pana `zero`
Izvi zvakakosha kunzwisisa
:::
Iwewe zvawaane vanhu 6 mukombi, uchati vanhu 6 ava vakagara pa masiti 6. Tarisa `list` rako iri:

```
vanhuMukombi = ["Panashe", "Tanaka", "Kupa", "Vimbai","Ruvarashe","Tatenda"]
```
:::info Iwewe unoverenga masiti evanhu seizvi
- `"Panashe"` aripasiti `1`
- `"Tanaka"` aripasiti `2`
- `"Kupa"` aripasiti `3`
- `"Vmbai"` aripasiti `4`
- `"Ruvarashe"` aripasiti `5`
- `"Tatenda"` aripasiti `6`
:::

Asi `computer` ichatangira pana `zero` kwete pana `1` seizvi:

:::info `Computer` inoverenga masiti evanhu seizvi
- `"Panashe"` aripasiti `vanhuMukombi[0]`
    - zvichireva kuti aripasiti rechi`zero`
- `"Tanaka"` aripasiti `vanhuMukombi[1]`
- `"Kupa"` aripasiti `vanhuMukombi[2]`
- `"Vmbai"` aripasiti `vanhuMukombi[3]`
- `"Ruvarashe"` aripasiti `vanhuMukombi[4]`
- `"Tatenda"` aripasiti `vanhuMukombi[5]`
Aya ndiwo matarisiro anoita `computer` zvinhu zviri mu`list`. Unogona kutora chinhu mu `list` ogadzira `variable` kuti uone kuti chii chinoitika
:::

**Chii chikuitika pa line 2?**
:::info Tsananguro: Kupinda mu list
Tiri kuti `pane ___ mu ___ {`. Apa tine 2 ma `___`
1. `___` wekutanga anomirira `variable` inopinda mu `list` 
- variable inopinda mu `list` inonyorwa semanyorerwo anoitwa ma `variables` akataurwa mu _topic_ yema `variables`. Asi apa kutongonyora `variable` yekuti `munhu` ndiko kutoigadzira kwacho, saka `computer` inonzwisisa kuti apa wagadzira variable ichapinda mu `list` rako
2. `___` wechipiri anomirira zita re`list` rako raukuda kupinda mariri
3. Ma _curly brackets_ `{}` anoudza `computer` kuti kana yapinda mulist, ishandise `code` irimo muma _curly brackets_ aya



**Ko pana Line 3?**

`Line 3` arimuma _curly brackets_ saka zvinoreva kuti `computer` ichasandisa `code` iyi kana ichinge yapinda mu `list` rako re `vanhuMukombi`.
```
nyora("Hesi ", munhu)
```
Apa `nyora()` apihwa zvekunyora zviri 2: `"Hesi "` na `munhu`.
1. `"Hesi "` arimuma _quotes_, zvichireva kuti `mashoko` saka `computer` inonzwisisa kuti "Hesi " achanyorwa mu `terminal` sezvaari
2. `munhu` haasi mu ma _quotes_, zvichireva kuti haasi `mashoko`, uye haisi `nhamba`, saka `computer` inonzwisisa kuti `munhu` i `variable`
- `computer` ichatsvaga kuti `variable` inonzi `munhu` inopinda mu `list` yambogadzirwa here? Hongu, yagadzirwa pa `line 3`



**Kana computer yapinda mu`list`**

Kana `computer` yapinda mu list rako, inotanga pane chinhu chekutanga mulist rako. Saka apa inotanga pana `"Panashe"`. 
Kana computer iri pana `"Panashe"`, `variable` inonzi `munhu` ichachinja kuita `"Panashe"`. 
```
munhu = "Panashe"
```
Computer paichashandisa `line 3`
```
nyora("Hesi ", munhu)
```
Apa `munhu` ndi `"Panashe"`, saka mu `terminal` muchanyorawa kuti 
`Hesi Panashe`.

Kana `computer` yapedza kushandisa code iri muma _curly brackets_`{}` pana `"Panashe"`, ichaenda pane chinhu chiri kutevera mu `list` rako yoita sezvayaita pana `"Panashe"`. Saka variable `munhu` ichachinja kuita `munhu = "Tanaka"` yonyora `nyora("Hesi ",munhu)`, zvichingodaro kusvika yapedza nevanhu vese vari mulist rako

### Kubuda mulist 

Kana `computer` ichinge yapinda mu `list`, inotangira pane chinhu chekutanga kusvika pane chinhu chekupedzisira chiri mu `list` imomo. 

<Question text="Ko kana uchida kubuda mulist usati wapedza zvinhu zvese zvirimo " />

Unoshandisa `buda` kubuda mu `list`. Ngatomboti iwewe ukuda kumhoresa vanhu 3 chete varimukombi yako iya, obva wanyarara hako, nekuti haudi zvekunyanya kutaura taura.

:::note Ngatironge kuti tichaita sei?
1. Tichapinda mu`list`
- Apa tichashandisa `code` iya yekupinda mu `list`
2. Tichada kuverenga kuti tave pane munhu wechi `3` here
- apa tichashandisa `maths`, uchazoona zvedzimwe maths kumberi uku
3. Kana tave pane munhu wechi `3`, tichabva `tabuda` mu `list`
- apa tichashandisa `buda`
:::

:::info **Chii chikuitika apa?**

**Pa `line 2` na `line 3`**

- Tagadzira `variable` inonzi `nhambaYeVanhu` pa `line 2`. `Variable` iyi i `nhamba` yatichange tichiwedzera pakuverenga vanhu tapinda mu `list`. `nhambaYeVanhu` iyi ichange ichiwedzera na `1` kana tabva pane munhu wekutanga tichienda pane munhu wechipiri, zvichingodaro. 
- Tagadzira `variable` inonzi `nhambaYekubuda` pa `line 3`, bvatati `nhamba` iyi ndi 3. Tichada kuzo `enzanisa` `variable` iyi ne `variable` ye `nhambaYeVanhu` ichange ichiwedzera kubva pana 0 zvichienda mberi

**Pa `line 4`**
- Apa tanyora `code` yekupinda mu `list` re `vanhuMukombi`

**Pa `line 5` na `line 6`**
- Tikuda kuwedzera `nhambaYeVanhu` na `1` kana tave kufamba kubva pane chinhu chimwe ne chimwe mu `list`. 
- Patinopinda mu `list`, tinotanga tiri pane munhu wekutanga, saka apa tikuudza `computer` kuti `nhambaYeVanhu` ngaiwedzere na `1` kubva pana `0`, saka yave `1`
- Pa `line 6`, tikuudza `computer` kuti `nyora()` zvinhu 3:
    1. `"Hesi "` - aya mashoko anonyora wezvaari
    2. `munhu` - i `variable` yekupinda mu `list`, saka panonyorwa zita remunhu
    3. `nhambaYeVanhu` - i `variable` ichange ichiwedzera na `1` sekufamba kwatinenge tichiita kubva pane `munhu` wekutanga mu `list` tichienda mberi

**Pa `line 7` , `line 8`, na `line 9`**
- Apa tikushandisa `enzaniso` kuenzanisa `nhambaYeVanhu` na `nhambaYekubuda`, **kana** `nhambaYeVanhu` yawedzera kusvika pana `3` inobva yafanana na `nhambaYekubuda`, saka enzaniso iyi inoita `chokwadi`. 
- Kana `enzaniso` yaita `chokwadi`, `computer` inobva yashandisa `code` irimuma _brackets_ `{}` epa `kana`.
- `code` ichashandiswa ne `computer` kana `enzaniso` iri `chokwadi` ndi `buda`. Izvi zvinoreva kuti chibuda mu `list`, chingosiyana nezvasara kumberi
