# Ma Functions


## Ma function chii?

Ma _functions_ i `code` yaunogadzira kana uchida kuzoituma kuti `basa` kumberi. Nemamwe mashoko, tiri kuti `function` i `basa`. Ngatimboti iwewe ukuda kugadzira `basa` rino mhoresa munhu. Rinogadzirwa se izvi: 

<Image source="/img/maths4.jpeg" />


**Chii chikuitika apa?**


:::info Tsananguro
Kubva pana `line 1`, `line 2`, na `line 3`, tagadzira `basa` rinonzi `mhoresaMunhu`. 
- Zita re `basa` rinofana kunyorwa sekunge uri urikugadzira `variable`. 
- Pamberi pezita re `basa` panoiswa ma _brackets_ `()`, muma brackets imomo munogona kuiswa ma `variable` kana kuti munogona kusaiswa chinhu, sezvaukuona pa `code` iyi.


Pa `line 5`:
```
tangaBasa.mhoresaMunhu()
```
Apa tiri kutuma computer kuita `basa` ratagadzira rinonzi `mhoresaMunhu()`.Ma _brackets_ `()` haasiiwe.
:::

:::success Ita
Nyora `code` iyoyi mu `index.shona`, wochiita izvi:

1. mu `terminal`, nyora kuti: 
`node shona.js` wobaya `enter`

2. mu `terminal`, nyora kuti: 
`node index.js` wobaya `enter`
:::
Ukaita izvi, unofana kuona mu `terminal` makanzi

```
Hesi
``` 

Ngatimbobvisa `line 5` rakanzi `tangaBasa.mhoresaMunhu()`

:::success Ita 
1. mu `index.shona` dzima `line 5` rekuti `tangaBasa.mhoresaMunhu()`
2. mu `terminal`, nyora kuti: 
`node shona.js` wobaya `enter`

3. mu `terminal`, nyora kuti: 
`node index.js` wobaya `enter`
:::

Ukaita izvi, mu `terminal` **hamuna chinonyorwa**. 

Izvi zvinoitika nekuti wagadzira `basa` rekumhoresa munhu, asi hauna kuzoudza `computer` kuti itanga `basa` racho, saka `basa` hariitwe kana risina kutangwa na `tangaBasa.mhoresaMunhu()`

### `Basa` ne ma `variable`

Unogona kugadzira basa rinotora `variable` inozoshandiswa ne basa racho kana razotanga. Ngatomboti iwewe ukuda kugadzira `basa` rinomhoresa munhu asi ukuda kuzopa `basa` zita remunhu wekumhoresa. Unoita seizvi: 

<Image source="/img/maths5.jpeg" />

**Chii chikuitika apa?**

:::info Tsananguro
Apa ukugadzira `basa` rinonzi `mhoresaMunhu`, asi ukuudza `computer` kuti kana basa iri rave kutangwa, rinofana kupihwa `variable` inonzi `zita` ichazoshandiswa na `nyora("Hesi ", zita)`.
:::

:::success Ita 
Nyora `code` iyoyi mu `index.shona`, wochiita izvi:

1. mu `terminal`, nyora kuti: 
`node shona.js` wobaya `enter`

2. mu `terminal`, nyora kuti: 
`node index.js` wobaya `enter`
:::

Ukaita izvi, unofana kuona mu terminal kuti 

```
Hesi Panashe
```

:::danger
Ukada kuenda kutanga `basa` usina kuisa variable, `computer` ichakuudza kuti 

```
paita problem, basa rinonzi `mhoresaMunhu` 
ranga richifana kupihwa variable 1 asi iwewe waisa mavariables 0.

```
:::

## Basa rinoshanda ne nguva


### Mirira _ seconds
unogona kugadzira `basa` rinomirira kuzotanga kana papfuura ma `seconds` aunenge wataura iwewe

<Image source="/img/basa4.jpeg" />

**Chii chikuitika apa?**

:::info Tsananguro
Tangodzokorora kugadzira `basa` rinonzi `mhoresaMunhu` sezvataita pakutanga, asi patagadzira `basa` iri, taudza `computer` kuti `mirira 3 seconds` kuitira kuti kana `basa` iri razotangwa, rinomirira ma _seconds_ enguva `3` robva razotanga.
:::

:::success Ita 
Nyora `code` iyoyi mu `index.shona`, wochiita izvi:

1. mu `terminal`, nyora kuti: 
`node shona.js` wobaya `enter`

2. mu `terminal`, nyora kuti: 
`node index.js` wobaya `enter`
:::
Ukaita izvi, uchaona kuti mu `terminal` munombotanga musina chinhu kwenguva diki, mozonyorwa kuti 
```
Hesi
```
`Hesi` anyorwa padarika 3 _seconds_ dze nguva

### Dzokorora _ seconds

<Image source="/img/basa5.jpeg" />

**Chii chikuitika apa?**
:::info Tsananguro
Zvekare tagadzira `basa` reku `mhoresaMunhu`, asi apa tikuda kuti `basa` iri rirambe richidzokororwa kana papfuura 1 _second_ ye nguva. 
:::
:::success Ita 
Nyora `code` iyoyi mu `index.shona`, wochiita izvi:

1. mu `terminal`, nyora kuti: 
`node shona.js` wobaya `enter`

2. mu `terminal`, nyora kuti: 
`node index.js` wobaya `enter`
:::
