:::danger Izvi zvinoshanda mu VS Code kana kuti mu Dhaga
Kana ukushandisa application ye ShonaScript, jamba topic iyi, unozoiona ku
:::

Unogona kuda kunyora `code` inoita kuti munhu apindure mu `terminal`, sokuti unogona kuda kuti munhu anyore zita rake mu `terminal`, iwewe wozotora zita iroro oita zvaunoda naro zvakaita seku mhoresa munhu iyeye. Tarisa pazasi apa, uone kuti izvi zvinotwa sei:

Kuita izvi, tinoshandisa zvinhu 2:

:::info Tsananguro
**Line 1**
```
teerera mhinduro zita
```
Tiri kuudza `computer` kuti iteerere zvichanyorwa mu `terminal`. Kana yateerera zvichanyorwa, ichabva yagadzira `variable` inonzi `zita`. `zita` apa ndizvo zvese zvichanyorwa mu `terminal`.
 
**Kubva pa `line 3` kuzvika pa `line 5`**
Apa taakuudza `computer` kuti tora mhinduro yanyorwa mu `terminal` ine variable rekuti `zita`, obva wa `nyora` kuti `"Hesi ", zita`

:::

### Waakushanda kuzvitupa zvekare

Takambokupa `basa` rekuudza vanhu vauya kuzotora zvitupa kuti vaende ku _Office_ kunotora chitupa `kana` vaine makore 16 zvienda mberi, kana kuti vaende kumba `kana` vaine makore 15 zvichidzika _mutopic_ ye [enzaniso](/docs/guide/conditionals). Patanyora `code` yezvitupa, takagadzira `variable` inonzi `makoreEmunhu` tikabva taipa makore acho iri `nhamba`. Kana wakanganwa, tarisa pazasi apa `code` yacho yese yatakanyora

<Image source="/img/kana2.jpeg"/>


<Question text="Unoita sei kana uchida kushandisa mhinduro kuwana makore emunhu"/>

**Chiita uchishandisa `mhinduro`**

Apa taakuda kunyora `code` iyi zvekare, asi ikozvino pane kuti ugadzire variable yekuti `makoreEmunhu` sezvawakaita pa `enzaniso`, uchagadzira variable iyoyo uchishandisa `teerera mhinduro` seizvi:

```
teerera mhinduro makoreEMunhu
```

`computer` ichamirira kuti makore emunhu anyorwe mu `terminal`, iwewe wobva watora mhinduro iyoyo ochishandisa `code` iri kubva pa`line 4` kusvika pa `line 8` mu _picture_ iri pamusoro apo. Kana wapedza, code yako inofana kuita seizvi:

<Image source="/img/mhinduro2.jpeg" />

:::success Ita
Nyora `code` iyoyi mu `index.shona`, wochiita izvi:

1. mu `terminal`, nyora kuti: 
`node shona.js` wobaya `enter`

2. mu `terminal`, nyora kuti: 
`node index.js` wobaya `enter`

```
mhinduro: makoreEMunhu: 
```


Izvi zvinoreva kuti `computer` yakamirira iwewe kuti unyore `makoreEmunhu`. Saka nyora kuti `nhamba` yemakore acho,  wobva wabaya `enter`.

Ukanyora `nhamba` iri pasi pa 16 sokuti 12, mu`terminal` munofana kunyora kuti: 

```
Uchiri mudiki, unozodzoka waane makore 16
```
Asi ukanyorawo `nhamba` iri pamusoro pa 16 sokuti 21, mu`terminal` munofana kunyorwa kuti:
```
Enda ku Office kunotora chitupa
```
