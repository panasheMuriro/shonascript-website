# Teerera mhinduro

Paunonyora message pa whatsapp, pane chinhu chinotarisa zvaukunyora. 
Paunozopedza kunyora wobaya `bhatani` re kutumira message yako, chinhu ichi chinozotora zvese zvawanyora chotumira semessage kune munhu wauri kutarura naye.


Kuti tiburitse pekunyorera pacho pa website yedu, tinoshandisa `tm`. Tinenge tichiedza kuti `teerera mhinduro` ichanyorwa ne munhu pa website.
Unokwanisa kuchinja mamiriro a `tm` sekungochinja kwaunoita `box`

:::success Zvekuita
Tikuda kuisa `tm` pa website yedu kuti tinyore zvatinoda
:::
