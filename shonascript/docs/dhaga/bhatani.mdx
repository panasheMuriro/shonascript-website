# Bhatani


`Mabhatani` anoshandiswa kana uchida kuti chimwe chinhu chiitike mushure mekunge munhu abaya pa `website` yako.
`Mabhatani` anowanikwa pama `website` akawanda uye muma `application` anoshanda muphone yako.

Unoisa mabhatani uchishandisa `bt` seizvi:

```
bt:
;
```

Uchazoziva zvakawanda zvinonakidza zvema bhatani kana tave kushandisa `ShonaScript` mu `Dhaga`.`ShonaScript` ndiyo ichange yaakuita kuti zvinhu zvichinje mushure mekunge munhu abaya bhatani pa website yako.
Hapana musiyano wakanyanya pakati pe `box` ne `bhatani`, asi chine basa ndechekuziva kuti `box` rinoisirwa zvinhu zvese asi `bhatani` rinoiswa mashoko kana `icon`. 
Uchazoona kuti ma `icon` chii pa topic yema `icon`.

Zvese zvaunokwanisa kuchinja pamamiriro e `box` unokwanisa kuchinjawo pama miriro e `bhatani`.

:::info chii chikuitika apa
- Taumba `bhatani` tichishandisa `bt`
    - tachinja `color ye bhatani`,`color yemashoko`, `makona`, `kufara`, ne `kureba` kwakaita sekuchinja kwatinongoita pa `box`
- `Bhatani` rako iri ukaribaya hapana zvinoitika nekuti harishande kana pasina code ye ShonaScript. Uchazoona kumberi kuti tinoshandisa `ShonaScript` mu `Dhaga` sei 
:::

Mazuva ano mamwe mabhatani aunoona pama website kana pamaapplication anenge aine ma `icon` , asina kunyorwa chinhu mukati.
Application ye ShonaScript ine mabhatani aunoshandisa aye kana wapedza kunyora `code`, kana kuti waakudzokera kunonyora `code`. 
Ukamatarisa ma `bhatani` aya, ane ma `icon` asi haana kunyorwa chinhu mukati.
Tichaedza kumagadzira ma bhatani aya pane `topic` iri kutevera
