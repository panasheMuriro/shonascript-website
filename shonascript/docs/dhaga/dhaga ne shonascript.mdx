# Dhaga ne ShonaScript

Apa ndopauchaona zvinhu kunyora `code` kwave kuita kunge `magic`, saka gadzirira.


## Box ne Bhatani
Tave kudzokera ku `box` ne `bhatani`. Taona kuti bhatani ne box zvinogadzirwa sei kumashure, asi ma bhatani edu asinga asinga shande nekuti haana kupihwa zvekuita zvacho kana munhu achinge amabaya. 
Taakuda kuumba zvinhu izvi:
1. `box` re `green`
2. `bhatani` rekutanga rekuti munhu akabaya, `box` rinochinja kuita `red`
3. `bhatani` rechi 2 rekuti munhu akabaya, `box` rinochinja kuita `green`
Tikuda kuedza kuteedzera zvinoita `traffic light` muroad richimisa mota 

Kuti Uzoudza ShonaScript zvekuita nema `box`, kana munhu achinge abaya ma `bhatani` unofana kupa zvinhu zvese izvi mazita kunge `mavariables`. 
Panguva ino unotarisirwa kunge uchiziva kuti `mavariables` anobvumidzwa akaita sei sezvakatsanangurwa mu ShonaScript. Saka tichapa mazita aya:
- `box` - boxOne
- `bhatani` rekutanga - btOne
- `bhatani` rechi 2 - btTwo

zita racho rinongonyorwa pamberi pe chinhu chacho sokuti:
```
box boxOne:
;
```

Tichabva taudza shonascript kuti kana munhu abaya bhatani rekutanga `btOne`, chinja `color` ye `boxOne` kuita `green`. Pakubayawo bhatani `btTwo`, chinja `color` ye `boxOne` kuita `red`

Unoshandisa shonascript mudhaga seizvi:
```
shonascript{

}
```
Izvi ndizvo zvinoratidza kuti waakuisa shonascript ichashanda ne `Dhaga` iri muma `brackets` ayo.

- Tanyora ShonaScript inoita kuti kana munhu abaya `btOne`, `box` riite green, asi kana abayawo `btTwo`, `box` riite `red`


**ShonaScript yashanda apa**
ShonaScript yatiri kushandisa apa ndiyo yekare, asi yakawedzerwa zvimwe zvinhu zvinoita kuti ugone kuishandisa ne `Dhaga`. 
Ngatitarise ShonaScript yatashandisa apa.

```
pakubaya btOne {

}
```
Apa tinenge tichiudza ShonaScript kuti _pakubayiwa kwa `btOne`_, ita izvi zviri muma brackets

```
pakubaya btOne {
    pa boxOne chinja c kuita "red"
}
```
Apa tikudza ShonaScript kuti kana munhu abaya `btOne`, enda pana `boxOne`, obva wachinja `c`, apa `c` ndiyo `color` iye ye `box`, kuita `red`.
Ukasaisa ma `quotes` pana `red`, ShonaScript ichafunga kuti red wacho i`variable`, saka ichaedza kutsvaga kuti `variable` yacho yambogadzirwa here mu shonascript. Kana variable iyi isina kumbogadzirwa, unoudzwa kuti `paita problem`.

::::

### Zvimwe Zvaunogona kuchinja

Pamusoro apa tangochinja `color` tichishandisa `c`
Unogona kuchinjawo izvi:

|Chekuchinja|Muenzaniso|
|---|---|
|c - color |pa boxOne chinja c kuita "red" |
|ms - mashoko | pa boxOne chinja ms kuita "Zita rako" |
|kn - kona | pa boxOne chinja kn kuita 4 |
|kf - kufara(width) | pa boxOne chinja kf kuita 2 |
|kr - kureba(length) | pa boxOne chinja kr kuita 3 |
|ms-c  - color yemashoko | pa boxOne chinja ms-c kuita "purple" |
|v  - viga `box` | pa boxOne chinja v kuita "hongu" kana kuti  pa boxOne chinja v kuita "kwete" |

Chine basa apa kuziva zvaukuda kuchinja pa `box` rako, hazvireve kuti unofana kuchinja zvinhu zvese zvepa `box`

###  Kuona zvinhu zvebox

Taona kuti unogona kuchinja zvinhu zvepa box kana zviri mukati macho uchishandisa 
```
pa boxOne chinja c kuita "red"
```
Unogon kunge uchida kuona kuti `c` wacho wepa `box` ndiani. Apa unenge uchiedza kumutora `c` iyeye woita zvaunoda naye, seizvi

```
pakubaya btOne{
    x = pa boxOne tora c
    nyora(x)
}
```

Apa tatora c pa bx one tichishandisa `pa boxOne tora c`, tobva tagadzira `variable` yacho yezvatinowana ipapo.
Pazasi pacho tazotuma shonascript kuti ichinyora `variable iyi`

### Kutora zviri emu box
Unokwanisa kutora mashoko arimu`box` rinonzi `boxOne` uchishandisa `ms` seizvi:

```
pa boxOne tora ms
```
Kana mubox muinewo nhamba chete pasina chimwe chinhu, unoitora nhamba iyoyo uchishandisa `nm` seizvi:

```
pa boxOne tora nm
```

Musiyano ur pakati pemashoko nenhamba ndewekuti mashoko haakwanise kusanganiswa sezvinoita ma nhamba sokuti `1+1` inokupa `2`
Ukaedza kusanganisa mashoko sokuti 
```
"panashe" + "muriro"
```
panobuda `panashe muriro`, mashoko aya haana kusanganisw, asi abatanidzwa, saka ndiwo musiyano uripo

Saka izvi zvinoreva kuti ukaenda mu shonascript ukanyora
```
"2"+"2"
```
panobuda `22`, hazvisi kureva `maths` yekuti kuti `2+2` inoburitsa `22`, asi zvikureva kuti mashoko ekuti "2" na "2" vabatanidzwa kuita `22`

Izvi zvakakosha kunzwisisa nekuti kana uine nhamba mu `box` sokuti
```
box boxOne:
23
;

```
obva waudza `ShonaScript` kuti tora mashoko arimu box uchishandisa
```
x = pa boxOne tora ms
```
ShonaScript inokupa x ari "23" apa, zvichireva kuti ukazoedza kuda kusanganisa x uye ne imwe nhamba sokuti 20
ShonaScript inotarisa zvinhu izvi sokuti
```
"23"+20
```
Nekuti ukuda kuedza kusanganisa mashoko ne nhamba, ShonaScript ichangobatanidza zvinhu izvi kuita `23 20` nekuti haukwanise kusanganisa `mashoko` ne nhamba

### Chinonzi Website ne Web Application

Kumashure uku tanga tichinyanya kutaura kuti `website`. 
Apa nekuti taakushandisa ma bhatani tichiita kuti munhu abaye achichinja zvinhu, tinenge tichigadzira inonzi `web application`. 
Musiyano uripo pakati pe `website` ne `web application` ndewekuti `website` inonyanya kunge iri yekuverenga zvinhu pasina zvakawanda wanda zvekubaya kana kuchinja, asi `web application` inoita mabhatani akawanda uye pane kuchinja chinja kunenge kuitika ipapo.
Kana ukaenda pa google wotanga kushandisa application ye`ShonaScript`, unenge uchisahndisa `web application` ipapo. Whatsapp haisi `web application`, itongori `application` yemuphone chaiyo. 
Makhambani anoshandisa ma `website` kuti vaudze vanhu basa ravanoita uye kuti kuratidza ma `picture` ebasa ravo.

`Web application` yakaoma kugadzira kupfuura `website`.

Ukaona uchinyora `code` ye `Dhaga` pasina `ShonaScript`, unobva waziva kuti ukugadzira `website`. Ukazotanga kunyora `ShonaScript` mu `Dhaga`, unobva waziva kuti ukugadzira `web application`

## Chii chinonzi bhegi?
Kuti unzwisise chinonzi `bhegi` mu ShonaScript, ngatimboti iwewe une list remazita evanhu seizvi:
```
zvinhu = ["madomasi", "zumbani","mango"]
```
Dambudziko riripo pa `list` rakadai nderekuti ungorine nemazita ezvinhu zvaukuda kunotenga pamusika chete pasina zvimwe. Ngatimboti iwewe ukuda kuti `list` rako riratidze mari inokwana kutenga chinhu chimwe nechimwe chiri mu list rako sokut:
```
madomasi anoita $3
zumbani rinoita $1
mango dzinoita $2
```
Kana ukada kushandisa `list` remashoko, zvinonetsa kuti uronge zvinhu izvi nemari yazvo. Apa ndipo panoshanda `bhegi`.
`Bhegi` ririkutaurwa nezvaro apa ndiwo ma`bhegi` aunoziva. Chine basa kuziva apa ndechekuti `bhegi` rine `homwe`. 
`Bhegi` rinogona kuita `homwe` dzakasiyana siyana, dzepakati, dzepaside, dzekunze zvichingodaro.
Isusu kana tichifunga nezvebhegi, tiri kuda kupa homwe dzebhegi mazita akasiyana siyana sokuti tichapa imwe homwe dze`bhegi` redu mazita ekuti `zita` ne `mari`.
Muhomwe inonzi zita, tichaisa zita rechinhu chikuda kunotengwa chacho, tozoisa mari ichashandiswa mu homwe inonzi `mari`.

Unoita izvi muShonaScript sokudai:
```
{"zita": "madomasi", "mari": 3}
```
Uku ndiko kutogadzira bhegi rine homwe yekutanga inonzi `zita` ine zita rechinhu chikuda kutengwa, yozoita imwe homwe inonzi `mari` ine nhamba yamari yacho ichashandiswa.
Kana waakuda kuona zviri muhomwe inonzi `zita`, unotanga wagadzira variable ye bhegi iri seizvi:
```
x = {"zita": "madomasi", "mari": 3}
```
Apa tiri kuti `x` ndiro bhegi redu ratagadzira, tinovhura homwe inonzi zita seizvi:

```
x["zita"]
```
Kuti tione kuti chiinobuda kana tavhura homwe iyi, tinenge tave kugadzira `variable` yatinozonyora kuti tione kuti zvii zvikuitika.
Taakudzokera ku ShonaScript

:::info chii chikuitika apa
- Apa tadzokera ku ShonaScript
    - baya `bhatani` riri pazasi pe screen ku `right`, woenda pakanzi `ShonaScript`
- Tagadzira bhegi rinonzi `x` rine homwe inonzi `zita` ine `"madomasi"`, ne imwe homwe inonzi `mari` ine `3`
- Tavhura homwe inonzi `zita`, tobva tagadzira `variable` yezvinhu izvi inonzi `y`
- Tavhura homwe inonzi `mari`, tobva tagadzira `variable` yezvinhu izvi inonzi `z`
- Tanyora ma `variables` aya kuti tione kuti zvii zvinobuda
    - `y` aburitsa `"madomasi"` ndiyo homwe yezita yatavhura ne `x["zita"]`
    - `z` aburitsa `3` ndiyo homwe yezita yatavhura ne `x["mari"]`
:::


Saka pane kuti uve ne `list` remashoko, unogona kuva ne `list` rema`bhegi` sokuti:

```
zvinhu = [
    {"zita": "madomasi", "mari": 3},
    {"zita": "zumbani", "mari":1},
    {"zita": "mango", "mari": 2}
]
```
Kana uine list rakadai, unogona kupinda mariri uchishandisa `pane chinhu mu zvinhu` sokuti:

```
pane chinhu mu zvinhu{

}
```
