#### Calculator 2

Taona kuti unogona kugadzira calculator inongosanganisa ma nhamba 2 sei. Taakuda kugadzira imwe calculator inoweka zvainenge yapihwa tichishandisa `nhamba.weka()`.
`nhamba.weka()` ano weka maths iri muma brackets odzosa mhinduro ye maths yacho. sokuti

```
nhamba.weka(2+3)
```

apa panobuda 5

```
nhamba.weka("2+3")
```

Apa panobuda 5 zvekare.
Unonogona kubvunza kuti `nhamba.weka(2+3)` haana kufanana nekungoti `2+3` here? 
Chine basa apa kuziva kuti `nhamba.weka()` anonyanya kushandiswa kana uchida kuweka maths dziri mumashoko sokuti `"2+3"`.
:::info Zvekuita mu ShonaScript

- Enda ku ShonaScript
- nyora izvi:

```
y = "2+3"
nyora(y)
```

Unoona kuti izvi hazvishande, saka kuti uweke maths iri mumashoko ekuti `"2+3"` unoshandisa `nhamba.weka()`.
:::

### Fembera nhamba
Tikuda kugadzira gemhu yekufembera nhamba. Tichada kuti ShonaScript isarudze nhamba iri pakati pa 0 na 3, tichishandisa `nhamba.chero(4)`. 
Iwewe uchanyora nhamba mu teerera mhinduro iri pakati pa 0 na 3.
Uchabaya bhatani kana wapedza. 
Pakubaya bhatani iri, uchaenzanisa nhamba yasarudzwa ne ShonaScript, ne yawanyora iwewe, kana zvakafanana, uchanyora kuti `wagona` mu box, `zvimwe` uchanyora kuti watadza.
Tinogona kugadzira rimwe box rekuzonyora nhamba ya sarudzwa ne ShonaScript, kuti uone kuti zveshuwa wagona here kana kuti watadza.

:::success zvekuita

- Gadzira box rine izvi:
  - zita rekuti boxOne
  - `rg-c`
  - `kf-10`
  - `kr-10`
  - `ms-c-white` color yemashoko white
- Gadzira rimwe box rine izvi:
  - zita rekuti `boxTwo`
  - `rg-c`
  - `kf-10`

- Gadzira teerera mhinduro `tm` ane izvi:
  - zita rekuti `tmOne`
  - `kf-10`
  - `color` yaunoda
- Gadira bhatani `bt` rine izvi:
  - zita rekuti `btOne`
  - `kf-2`
  - `kr-2`
  - `color` yaunoda
  - icon yepa nhamba 72 ine color ne size yaunoda sokuti `ic-72-white-3`

**Mu ShonaScript**
- Tikuda kuti munhu atanga anyora nhamba yake mu `tmOne`, obva abaya bhatani.
- Pakubaya bhatan iri, tikuda kuti ShonaScript isarudze nhamba, wobva watora nhamba yasarudzwa ne ShonaScript, ne nhamba yanyorwa mu `tmOne` woma enzanisa
- Kana ma nhamba aya akaenzana, nyora mu `boxOne` kuti wagona, zvimwe nyora kuti watadza.
- Tikuda kuratidza nhamba ya sarudzwa ne ShonaScript pa web application yedu, saka tichaisa nhamba iyoyo mu `boxTwo`

```
pakubaya btOne{
x = pa tmOne tora nm
y = nhamba.chero(4)

pa boxTwo chinja ms kuita y
kana x akaenzana na y {
pa boxOne chinja c kuita "green"
pa boxOne chinja ms kuita "Wagona"
}

zvimwe{
pa boxOne chinja c kuita "red"
pa boxOne chinja ms kuita "Watadza"
}

}
```

:::
